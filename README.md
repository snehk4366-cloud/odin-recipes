# Project: Recipes (The Odin Project)

A fundamental multi-page HTML project designed to practice the absolute essentials of web development. 

Since this project focuses entirely on structural HTML, it won't have any flashy styling just yet. I will be returning to this repository later in the curriculum to breathe life into it using CSS.

###  Skills Demonstrated:
* **HTML Boilerplate & Structure:** Setting up proper `<!DOCTYPE html>`, `<head>`, and `<body>` tags across multiple files.
* **Text Formatting:** Utilizing structural headings (`<h1>`, `<h3>`) and paragraphs (`<p>`).
* **Navigation:** Using anchor tags (`<a>`) to link relative paths between a root homepage and a sub-directory.
* **Data Organization:** Implementing unordered lists (`<ul>`) for ingredients and ordered lists (`<ol>`) for sequential recipe steps.
* **Media:** Embedding images using proper source paths and alternative text attributes.
* **Git Workflow:** Practicing atomic commits and pushing code to a remote GitHub repository.
